toDouble

convert symbolic to double

Calling Sequence

d = toDouble( s )

Arguments

s

symbolic number matrix

d

double matrix

Description

Converts symbolic number matrix s to a Scilab double matrix d.

Examples

// matrix of symbolic numbers
s = Sym("sqrt(pi)+sqrt(2)")(ones(2,3))

// convert to Scilab double
d = toDouble(s)
