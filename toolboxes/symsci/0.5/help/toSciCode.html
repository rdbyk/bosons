toSciCode

convert symbolic to Scilab code

Calling Sequence

b = toSciCode( a )

Arguments

a

symbolic matrix

b

string matrix

Description

Converts a symbolic matrix a to a string matrix b of corresponding Scilab code.

Examples

// symbols
Syms f g x

// symbolic expression f ...
f = (x + x * Exp(x))^int32(2)

// ... and its derivative
g = Diff(f,x)

// both as Scilab code
toSciCode(f)
toSciCode(g)
