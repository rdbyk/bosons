LUSolve

solve symbolic linear system

Syntax

x = LUSolve( A, b )

Arguments

A

symbolic (n,n)-matrix

b, x

symbolic (n,1)-matrix

Description

Returns the solution x of the linear system A*x = b.

Examples

// setup matrices A, b 
A = Mat("a",2,2)
b = Mat("b",2,1)

// solution of A*x = b
LUSolve(A,b)
