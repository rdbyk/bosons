isNotEqual

(~=) symbolic non-equality relation

Syntax

t = isNotEqual( a, b )
t = ( a ~= b )

Arguments

a, b

symbolic

t

boolean

Description

Returns true (%t) if symbolic objects a and b are equal and false (%f) otherwise.

Examples

// some symbolic stuff
a = Sym("x+5")
b = Sym("5+x")
c = Sym("1")
d = Sym("1.0")

// symbolic equality?
a   ~= b
b   ~= c
c   ~= c
2*c ~= 2*d
c   ~= d
