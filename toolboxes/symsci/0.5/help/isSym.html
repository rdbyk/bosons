isSym

symbolic predicate

Syntax

t = isSym( s )

Arguments

s

any type

t

boolean

Description

Returns true (%t) if s is symbolic and false (%f) otherwise.

Examples

// some variables
a = 1.234
b = zeros(5,5)
c = Sym("(x+y)*(x-y)^10")

// what is symbolic?
isSym(a)
isSym(b)
isSym(c)
