DivElem

(./) symbolic elementwise division operator

Syntax

c = DivElem( a, b )
c = a .* b

Arguments

a, b

symbolic, double, or integer matrix

c

symbolic matrix

Description

Returns the elementwise quotient c = a ./ b.

It is mandatory, that at least one of the operands a or b is symbolic. Double and integer arguments are converted automatically to a symbolic representation, before the operation is carried out.

Examples

// create matrices
A = Mat("a",2,2)
B = Mat("b",2,2)

// elementwise quotient of matrices
A./B
