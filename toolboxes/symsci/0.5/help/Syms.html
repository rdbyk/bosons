Syms

create symbolic variables

Calling Sequence

Syms a1 a2 ... an

Arguments

a1, a2, ..., an

string

Description

Creates Scilab variables named a1, a2, ..., an whose values are symbols "a1", "a2", ..., "an".

Please note, a priori existing variables a1, a2, ..., an will be lost.

Examples

// create a single variable x
Syms x // i.e. x = Sym("x")

// create several variables at once
Syms a b c d

// use the variables
a = 2*x + 5
b = a/2
c = (a+b)^3
d = c-a
