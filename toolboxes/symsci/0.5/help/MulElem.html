MulElem

(.*) symbolic elementwise multiplication operator

Syntax

c = MulElem( a, b )
c = a .* b

Arguments

a, b

symbolic, double, or integer matrix

c

symbolic matrix

Description

Returns the elementwise product c = a .* b.

It is mandatory, that at least one of the operands a or b is symbolic. Double and integer arguments are converted automatically to a symbolic representation, before the operation is carried out.

Examples

// create matrices
A = Mat("a",2,2)
B = Mat("b",2,2)

// elementwise product of matrices
A.*B
