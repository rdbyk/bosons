rows

number of rows of a symbolic matrix

Syntax

c = rows( A )

Arguments

A

symbolic matrix

c

non-negative integer

Description

Returns the number of rows r of a symbolic matrix A.

Examples

// create a (3,4)-matrix
M = Mat("e"(ones(3,4)))

// number of rows
rows(M)

// number of cols
cols(M)

// number of elements
elems(M)
