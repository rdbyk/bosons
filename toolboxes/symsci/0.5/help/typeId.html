typeId

type id of a symbolic

Calling Sequence

id = typeId( a )

Arguments

a

symbolic

id

non-negative integer

Description

Returns the type (id) of a symbolica.

Examples

// some symbolic scalars ...
a = Sym("x+5")
b = Sym("5+x")
c = Sym("1")
d = Sym("1.0")

// ... and their type ids

typeId(a)
typeId(b)
typeId(c)
typeId(d)
