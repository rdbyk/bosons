elems

number of elements of a symbolic object

Syntax

n = elems( s )

Arguments

s

symbolic

n

non-negative integer

Description

Returns the number of elements n of a symbolic object s.

Examples

// create a vector
v = Vec("x",5)

// create an empty set
S = Set()

// create a matrix with diagonal v
M = Diag(v)

// number of elements in v, S, and M
elems(v)
elems(S)
elems(M)
