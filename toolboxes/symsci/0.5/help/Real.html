Real

real part of symbolic complex number

Syntax

b = Real( a )

Arguments

a, b

symbolic

Description

Returns the real part b = Real(a) of a.

Examples

// complex numbers ...
z1 = Sym("1+2*I")       // Complex
z2 = Evalf(z1, 0, %t)   // ComplexDouble
z3 = Evalf(z1, 200, %t) // ComplexMPC


// .. and their real parts
Real(z1)
Real(z2)
Real(z3)
