Fraction

symbolic expression as fraction

Syntax

[ n, d ] = Fraction( e )

Arguments

e, n, d

symbolic

Description

Rewrites an expression as fraction e = n/d and returns its numerator n and the denominator d.

Examples

// create an expression
e = Sym("1/x+x/y+z/x")

// rewrite expression as fraction
[n,d] = Fraction(e)
   
   // rebuild expression from n and d
n/d
