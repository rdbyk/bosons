Transpose

(') symbolic transpose operator

Syntax

B = Transpose( A )
B = A'

Arguments

A, B

symbolic matrix

Description

Returns the tranposition B = A' of A.

Examples

// create matrix
A = Mat("a",2,3)

// transposed matrix
A'
